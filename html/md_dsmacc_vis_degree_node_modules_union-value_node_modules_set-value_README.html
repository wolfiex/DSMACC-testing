Create nested values and any intermediaries using dot notation (‘'a.b.c’`) paths.

Install

Install with npm:

$ npm install --save set-value

Usage

var set = require('set-value');
var obj = {};
set(obj, 'a.b.c', 'd');
console.log(obj);
//=> {a: {b: c: 'd'}}
