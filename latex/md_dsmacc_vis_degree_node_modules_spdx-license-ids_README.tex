Type\+: {\ttfamily string\mbox{[}\mbox{]}}

All license I\+Ds except for the currently deprecated ones.

//=> ['0BSD', 'AAL', 'ADSL', 'AFL-1.1', 'AFL-1.2', 'AFL-2.0', 'AFL-2.1', 'AFL-3.0', 'AGPL-1.0-only', ...]
