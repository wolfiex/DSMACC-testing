\section*{home-\/path}

Cross-\/platform home directory retriever, tested on Windows XP and above, Mac O\+SX and Linux.

With node versions 2.\+3.\+0 (iojs) or higher, the built-\/in \href{https://nodejs.org/api/os.html#os_os_homedir}{\tt {\ttfamily os.\+homedir}} method is used.

{\bfseries Example}
