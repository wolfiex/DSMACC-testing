\subsection*{parents(dir, opts)}

Return an array of the parent directories of {\ttfamily dir}, including and starting with {\ttfamily dir}. If a {\ttfamily dir} isn\textquotesingle{}t specified, {\ttfamily process.\+cwd()} will be used.

Optionally specify an {\ttfamily opts.\+platform} to control whether the separator and paths works the unixy way with `'/\textquotesingle{}{\ttfamily or the windowsy way where sometimes things use }\textquotesingle{}/\textquotesingle{}{\ttfamily and sometimes they use}\textquotesingle{}\textbackslash{}\textquotesingle{}{\ttfamily and also there are leading drive letters and other exotic features. If}opts.\+platform$<$tt$>$isn\textquotesingle{}t specified,process.\+platform$<$tt$>$ will be used. Anything that matches/$^\wedge$win/\`{} will use the windowsy behavior.
